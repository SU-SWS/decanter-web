<Section>
[Decanter v7](https://github.com/SWS/decanter/tree/v7) customizes Tailwind CSS for you to generate utility classes that works with our design system. Before you begin, please read and review the installation process on the official [Tailwind CSS website](https://tailwindcss.com/docs/installation). Please also become familiar with [Tailwind CSS presets](https://tailwindcss.com/docs/presets) as Decanter uses this functionality to make it even easier for you to use.
</Section>

<Section heading="Frameworks">
If you are adding Decanter to an existing project please have a review of the [Framework Guides](https://tailwindcss.com/docs/installation/framework-guides) available from Tailwind CSS.

1. **Install Tailwind into your framework of choice.**

    Some options include:

    [Next.js](https://tailwindcss.com/docs/guides/nextjs) | [Nuxt](https://tailwindcss.com/docs/guides/nuxtjs) | [SvelteKit](https://tailwindcss.com/docs/guides/sveltekit) | [Angular](https://tailwindcss.com/docs/guides/angular) | [Vite](https://tailwindcss.com/docs/guides/vite) | [Laravel](https://tailwindcss.com/docs/guides/laravel) | [Symfony](https://tailwindcss.com/docs/guides/symfony) | [Astro](https://tailwindcss.com/docs/guides/astro) | [Create React App](https://tailwindcss.com/docs/guides/create-react-app)

2. **Add Decanter as a dependency through your favorite package manager.**
    ```bash
     // Bash, shell, zsh, fish, etc.
      yarn add -D decanter
      npm install -D decanter
      pnpm install -D decanter
    ```

3. **Add Decanter to your tailwind.config.js file as a preset.**
    ```js
     // tailwind.config.js
      module.exports = {
        ...
        presets: [
          require('decanter')
        ],
        ...
      }
    ```

4. **Install fonts**
    It is recommended that you follow your framework's convention for installing fonts. You'll want to install Source Sans 3 and Source Serif 4 from Google Fonts and include the ligature font for the Stanford Logo if you are using it. Here is an example or file you can include in your build to install the fonts.

    https://github.com/SU-SWS/decanter/blob/main/fonts.css

    ```css
    @charset "UTF-8";

    /** Source Sans 3, Source Serif 4, Roboto Slab, Roboto Mono (regular only) from Google Font */
    @import url('https://fonts.googleapis.com/css2?family=Roboto+Mono&family=Roboto+Slab:wght@300;400;700&family=Source+Sans+3:ital,wght@0,400;0,600;0,700;1,400;1,600;1,700&family=Source+Serif+4:ital,wght@0,400;0,600;0,700;1,400;1,600;1,700&display=swap');
    @font-face {
      font-family: "Stanford";
      src: url("https://www-media.stanford.edu/assets/fonts/stanford.woff") format("woff"), url("https://www-media.stanford.edu/assets/fonts/stanford.ttf") format("truetype");
      font-weight: 300;
      font-display: swap;
    }
    ```

<Section heading="Standalone">

<div className="children:children:rs-mb-4">
1. **Install Decanter Tailwindcss**

    The easiest way to add Decanter to your project is to use the npm package manager.
    ```bash
     npm install -D decanter
      npx tailwind init
    ```

2. **Configure your tailwind.config.js file**

    Configure your project to use the Decanter preset of Tailwind CSS and point

    ```js
    module.exports = {
      // This preset option pulls in all of the Tailwind CSS configuration.
      presets: [
        require('decanter')
      ],
      // This should point to your source files as Tailwind CSS scans your files for the css classes you use.
      content: ["./src/**/*.{html,js}"],
      // You can still extend and mdify Tailwind CSS as you normally would.
      theme: {
        extend: {},
      },
      // You can still add plugins as you normally would.
      plugins: [],
    }
    ```

3. **Add the tailwind directives to your main CSS file.**

    `/src/css/main.css`

    ```css
    /** Source Sans 3, Source Serif 4, Roboto Slab, Roboto Mono (regular only) */
      @import url('https://fonts.googleapis.com/css2?family=Roboto+Mono&family=Roboto+Slab:wght@300;400;700&family=Source+Sans+3:ital,wght@0,400;0,600;0,700;1,400;1,600;1,700&family=Source+Serif+4:ital,wght@0,400;0,600;0,700;1,400;1,600;1,700&display=swap');

      @font-face {
        font-family: "Stanford";
        src: url("https://www-media.stanford.edu/assets/fonts/stanford.woff") format("woff"), url("https://www-media.stanford.edu/assets/fonts/stanford.ttf") format("truetype");
        font-weight: 300;
        font-display: swap;
      }

      @tailwind base;
      @tailwind components;
      @tailwind utilities;
    ```

4. **Compile using the Tailwind CSS CLI tool**

    Run the CLI tool to scan your template files for classes and build your CSS.

    ```sh
    npx tailwindcss -i ./src/css/main.css -o ./dist/css/styles.css --watch
    ```

5. **Add the compiled css to your project**

    Add your compiled CSS file to the `<head>` and start using Tailwind's utility classes to style your content.

    ```html
     <!doctype html>
      <html>
      <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <link href="/dist/css/styles.css" rel="stylesheet">
      </head>
      <body>
        <h2 class="type-1 font-bold underline">
          Hello world!
        </h2>
      </body>
      </html>
    ```

6. **Start developing**

    You can now use the Tailwind CSS classes available from Decanter and Tailwind CSS. Please continue reading about how to use Tailwind CSS in their [core concepts section](https://tailwindcss.com/docs/utility-first) and then come back and have a look at the examples and conventions sectioins in this website.
</div>
