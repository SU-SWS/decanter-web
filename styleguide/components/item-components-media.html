1.15 Components.Media Media

Toggle example guides Toggle HTML markup

A media element, e.g., image, video (self-hosted or embedded, e.g., YouTube), audio, iframe etc., with the option of including a caption and a link. To constrain the aspect ratio of the media element, add classes e.g., su-aspect-ratio--1x1 to the .su-media__wrapper container, or use the mixin @aspect-ratio to set custom aspect ratio on the .su-media__wrapper container. The default .su-media component takes custom HTML for the media content such as an iframe, e.g., YouTube embeds.
